# Wire up curik mcp CLI subcommand

## Description

real implementation that starts the MCP server from `curik.server`. Verify
`.mcp.json` template in `init_course` matches the actual invocation.

- [ ] `.mcp.json` generated by `init_course` correctly points to `curik mcp`
