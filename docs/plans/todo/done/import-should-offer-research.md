# Import process should offer research step

When importing/sequestering existing content, the flow currently goes straight
from content analysis into Phase 1a (course concept). It should offer a
research step — just like a new curriculum would get in Phase 1c.

After analyzing the old content, the agent should ask: "Would you like me to
research what's out there? I can look for related courses, standards, and
certifications that might inform how we structure this."

This is the same research the agent would do for a brand-new course, but
framed around the existing content: "Your course covers X, Y, Z — let me see
if there are certifications or established curricula that cover similar ground."

This should appear as an option in the post-sequester menu alongside "import
existing outline" and "start fresh."
