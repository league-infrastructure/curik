Welcome to **Web Development**, a hands-on course for grades 6–8.
You'll build real web pages from scratch using HTML and CSS[^1].

[^1]: HTML was created by Tim Berners-Lee in 1991. CSS was proposed by Håkon Wium Lie in 1994. Together they form the foundation of every website you've ever visited.

{{< callout type="info" >}}
This course assumes **no prior web development experience**. You should
be comfortable typing and using a text editor.
{{< /callout >}}

## What You'll Learn

- :globe_with_meridians: How to structure web pages with **HTML**
- :art: How to style and layout pages with **CSS**
- :iphone: How to make pages look great on any device with **responsive design**

## Prerequisites

- [ ] A modern web browser (Chrome, Firefox, or Edge)
- [ ] A text editor (VS Code recommended)
- [ ] Basic typing skills

{{< callout type="tip" >}}
Install the **Live Server** extension in VS Code — it will automatically
refresh your browser whenever you save a file!
{{< /callout >}}

## Course Structure

| Module | Topics | Estimated Time |

{{< instructor-guide >}}
**Course Overview for Instructors**

This 9-week course introduces web development fundamentals. Each lesson
includes exercises that students complete in their own `lessons/` directory.

**Key approach:** Always have students view their work in the browser
after every change. Immediate visual feedback is the most powerful
learning tool for web development.

**Assessment:** Students submit their completed lesson directories.
Each `README.md` contains the exercise instructions.
{{< /instructor-guide >}}
