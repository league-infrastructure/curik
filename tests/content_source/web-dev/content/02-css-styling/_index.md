Now that you can build HTML pages, it's time to make them look great
with **CSS** (Cascading Style Sheets).

{{< callout type="info" >}}
:art: **HTML = structure, CSS = style.** HTML tells the browser *what*
to show. CSS tells it *how it should look* — colors, fonts, spacing,
layout, and more.
{{< /callout >}}

## Module Goals

- [ ] Target elements with CSS selectors
- [ ] Control layout with flexbox
- [ ] Build responsive designs that work on any screen size

> "Design is not just what it looks like and feels like. Design is how
> it works." — Steve Jobs

{{< instructor-guide >}}
**Module Notes:** This module has three lessons covering selectors,
layout, and responsive design. Students should have completed the HTML
Basics module first.

**Pacing:**
- Selectors: 30 min
- Layout: 40 min
- Responsive: 35 min

**Key tip:** Use the browser DevTools extensively. Have students modify
CSS in the inspector before writing it in their files — this speeds up
the experimentation loop dramatically.
{{< /instructor-guide >}}
