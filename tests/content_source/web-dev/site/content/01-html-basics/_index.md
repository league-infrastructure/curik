In this module you'll learn the building blocks of every web page:
**HTML elements**. By the end, you'll be able to create structured,
well-organized web pages.

{{< callout type="info" >}}
HTML stands for **HyperText Markup Language**. It's not a programming
language — it's a *markup* language that tells browsers how to
structure content.
{{< /callout >}}

## Module Goals

- [x] Create your first HTML page
- [ ] Understand HTML elements and attributes
- [ ] Build a structured page with headings, lists, and links

| Tag | Purpose | Example |
|-----|---------|---------|
| `<h1>`–`<h6>` | Headings | `<h1>My Title</h1>` |
| `<p>` | Paragraphs | `<p>Some text.</p>` |
| `<a>` | Links | `<a href="...">Click</a>` |
| `<img>` | Images | `<img src="..." alt="...">` |
| `<ul>`, `<ol>` | Lists | `<ul><li>Item</li></ul>` |

{{< instructor-guide >}}
**Module Notes:** Start with a completely blank file so students see
exactly what a browser does with raw text (nothing!) vs structured HTML.

This "blank slate" approach is more effective than starting with a
template.
{{< /instructor-guide >}}
