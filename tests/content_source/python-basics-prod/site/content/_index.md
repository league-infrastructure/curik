Welcome to **Python Basics**, an introductory course for grades 3–5.
In this course you will learn the fundamentals of programming using Python[^1].

[^1]: Python was created by Guido van Rossum and first released in 1991. It is named after *Monty Python's Flying Circus*, not the snake :snake:.

{{< callout type="info" >}}
This course assumes **no prior programming experience**. If you have
coded before, you may want to skip ahead to [Control Flow](/02-control-flow/).
{{< /callout >}}

## What You'll Learn

- :package: How to store and use data with **variables**
- :twisted_rightwards_arrows: How to make decisions with **if statements** and repeat actions with **loops**
- :wrench: How to organize your code with **functions**

## Prerequisites

- [ ] Python 3.8 or newer installed
- [ ] A text editor (VS Code recommended)
- [ ] A terminal or command prompt

## Getting Started

Make sure you have Python installed on your computer. Open a terminal and type:

```bash
python3 --version
```

You should see something like `Python 3.11.4`. If not, ask your instructor
for help installing Python.

{{< callout type="tip" >}}
On Windows, you may need to use `python` instead of `python3`. Try both
and use whichever one works!
{{< /callout >}}

{{< instructor-guide >}}
**Course Overview for Instructors**

This course is designed for 8 sessions of 45 minutes each. Each lesson
includes a "Try It Yourself" section — make sure students have at least
10 minutes to work on these exercises independently.

**Key pedagogical approach:** Use concrete analogies (boxes, jars, recipes)
before introducing abstract concepts. Let students experiment and make
mistakes — debugging is learning.
