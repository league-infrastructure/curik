In this module you will learn what variables are, how to create them,
and the different types of data they can hold.

{{< callout type="info" >}}
Variables are one of the most fundamental concepts in all of programming.
Every program you write will use them!
{{< /callout >}}

## Module Goals

- [x] Understand what a variable is
- [x] Learn how to create and name variables
- [ ] Identify the three basic data types
- [ ] Convert between data types

## Key Vocabulary

| Term | Definition |
|------|-----------|
| **Variable** | A named container for storing data |
| **Assignment** | Giving a variable a value using `=` |
| **String** | Text data, wrapped in quotes |
| **Integer** | A whole number (no decimal point) |
| **Float** | A number with a decimal point |

{{< instructor-guide >}}
**Module Notes:** This module has two lessons. Lesson 1 (What Are Variables?)
should be taught with plenty of whiteboard examples. Lesson 2 (Data Types)
benefits from live coding demonstrations.

Spend extra time on the `type()` function — students often confuse the
*value* with the *type* of a value.
{{< /instructor-guide >}}
