Welcome to **Python Basics**, an introductory course for grades 3-5.
In this course you will learn the fundamentals of programming using Python.

## What You'll Learn

- How to store and use data with **variables**
- How to make decisions with **if statements** and repeat actions with **loops**
- How to organize your code with **functions**

## Getting Started

Make sure you have Python installed on your computer. Open a terminal and type:

```
python3 --version
```

You should see something like `Python 3.11.4`. If not, ask your instructor
for help installing Python.
